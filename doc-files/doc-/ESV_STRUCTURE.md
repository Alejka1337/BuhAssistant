# Структура файлов ЄСВ (ESV)

## Описание
Для упрощения парсинга все файлы `{Регион}ЄСВ.xlsx` должны иметь единую структуру.

## Требования к структуре файла

### Формат
- **Строка 1:** Заголовки (обязательно!)
- **Строка 2 и далее:** Данные

### Порядок колонок (фиксированный)

| Колонка | Название | Описание | Пример |
|---------|----------|----------|--------|
| A | Банк отримувача | Название банка | Казначейство України (ЕАП) м. Київ |
| B | Назва органу ДПС (отримувач) | Получатель | ГУ ДПС у Київській обл. |
| C | Код за ЄДРПОУ органу ДПС | ЄДРПОУ код | 44073213 |
| D | Номер рахунку (IBAN) | IBAN счета | UA838999980000355659201020232 |
| E | Символ звітності | 201 или 204 | 201 |
| F | Категорії платників | Описание категорий | Єдиний внесок нарахований за себе... |

### Правила:
1. **Символ звітності** должен быть только `201` или `204`
   - `201` = ЄСВ ФОП 2/3 группа
   - `204` = ЄСВ за працівників

2. Все пустые строки будут пропущены

3. Файл должен быть в формате `.xlsx` или `.xls`

## Пример файла

```
| Банк отримувача | Назва органу ДПС | Код за ЄДРПОУ | Номер рахунку (IBAN) | Символ звітності | Категорії платників |
|-----------------|------------------|---------------|---------------------|------------------|---------------------|
| Казначейство України | ГУ ДПС у Київській обл. | 44073213 | UA838999980000355659201020232 | 201 | Єдиний внесок нарахований за себе... |
| Казначейство України | ГУ ДПС у Київській обл. | 44073213 | UA658999980000355669204020232 | 204 | Єдиний внесок за працівників... |
```

## Что делать
1. Откройте каждый файл `{Регион}ЄСВ.xlsx`
2. Убедитесь, что колонки расположены в указанном порядке (A-F)
3. Убедитесь, что первая строка - это заголовки
4. Убедитесь, что в колонке E только значения `201` или `204` (без `.0`)
5. Сохраните файл

После этого новый парсер будет корректно обрабатывать все файлы.
